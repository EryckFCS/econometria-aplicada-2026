# Unidad 1: Técnicas y Modelos de Series de Tiempo

Esta unidad se enfoca en el análisis de variables económicas a través del tiempo, permitiendo estimar equilibrios de corto y largo plazo.

## Contenidos Teóricos

- **Modelos Dinámicos**: Inclusión de rezagos de la variable dependiente y de las independientes.
- **Modelos ARIMA**: Autoregressive Integrated Moving Average para pronóstico univariado.
- **Modelos Volátiles (ARCH y GARCH)**: Análisis de la varianza condicional (volatilidad) en series financieras y económicas.
- **Ecuaciones Simultáneas**: Modelado de interdependencias entre múltiples variables endógenas.

## Objetivos de Aprendizaje

- Estimar equilibrios económicos.
- Analizar la causalidad entre variables con sentido crítico.
- Utilizar técnicas de modelado dinámico para entender la persistencia de los fenómenos.
