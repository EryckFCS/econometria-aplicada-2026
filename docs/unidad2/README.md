# Unidad 2: Pronósticos y Simulación con Datos de Series de Tiempo

Esta unidad profundiza en el comportamiento de largo plazo y la capacidad predictiva de las series temporales económicas y sociales.

## Contenidos Teóricos

- **Raíces Unitarias (Unit Roots)**: Evaluación de la estacionalidad de las series (Test de Dickey-Fuller Aumentado, etc.).
- **Cointegración**: Estudio de las relaciones de largo plazo entre variables no estacionarias del mismo orden.
- **Modelos de Corrección de Error (MCE)**: Análisis de la velocidad de ajuste hacia el equilibrio de largo plazo.
- **Causalidad de Granger**: Determinación de si la información de una serie ayuda a predecir otra.
- **Pronósticos Temporales**: Simulación y proyecciones basadas en modelos econométricos robustos.

## Objetivos de Aprendizaje

- Realizar pronósticos precisos de variables económicas para la toma de decisiones.
- Proyectar trayectorias futuras con responsabilidad y rigor estadístico.
