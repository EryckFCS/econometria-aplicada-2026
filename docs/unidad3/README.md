# Unidad 3: Técnicas de Cointegración, Causalidad y Datos de Panel

Esta unidad integra el análisis de series de tiempo con datos transversales para un análisis regional y comparativo profundo.

## Contenidos Teóricos

- **Cointegración en Datos de Panel**: Extensión de los modelos de largo plazo para múltiples unidades (Kao, Pedroni, Westerlund tests).
- **Causalidad en Datos de Panel**: Determinación de relaciones direccionales en estructuras transversales dinámicas (Dumitrescu-Hurlin, etc.).
- **Revisiones de Datos de Panel**: Ajustes para heterocedasticidad, autocorrelación y dependencia de sección transversal.

## Objetivos de Aprendizaje

- Emplear técnicas avanzadas de panel para comprender fenómenos regionales y nacionales.
- Analizar la causalidad en estudios comparados internacionales.
